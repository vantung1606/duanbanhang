# Backend Skill Guide (`DIY`)

This folder contains the specialized skills for the Spring Boot backend. Use these skills to ensure consistent patterns and best practices.

## Available Skills

- **[springboot-patterns](file:///.agent/skills/springboot-patterns/SKILL.md)**: Core Spring Boot architecture, Java 21 features, MapStruct, and Controller/Service/Repository patterns.
- **[springboot-security](file:///.agent/skills/springboot-security/SKILL.md)**: JWT authentication, Spring Security configuration, and authorization patterns.
- **[jpa-patterns](file:///.agent/skills/jpa-patterns/SKILL.md)**: Spring Data JPA, entity design, and transaction management.
- **[mysql-patterns](file:///.agent/skills/mysql-patterns/SKILL.md)**: MySQL-specific optimization, indexing, and schema design.
- **[springboot-tdd](file:///.agent/skills/springboot-tdd/SKILL.md)**: Testing strategies with JUnit 5, Mockito, and MockMvc.
- **[springboot-verification](file:///.agent/skills/springboot-verification/SKILL.md)**: Quality assurance loop and static analysis.
- **[backend-patterns](file:///.agent/skills/backend-patterns/SKILL.md)**: General backend architectural patterns.
- **[database-migrations](file:///.agent/skills/database-migrations/SKILL.md)**: Managing schema changes with Liquibase/Flyway.

## How to Use
When working in the `DIY/` directory, refer to these skills to maintain high-quality code and follow established project standards.
